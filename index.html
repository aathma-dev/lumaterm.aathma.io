Your workspace,
your layout

With Multiple Sessions

Split panes vertically or horizontally. Zoom any pane to full screen. Navigate between panes with arrow keys. Each pane tracks its own working directory and shell.

⌘D Split vertical ⌘⇧D Split horizontal ⌘⇧↩ Zoom pane

Git, without
leaving the terminal

A full git panel built right in. Visual commit graph with branch lanes, file change tracking, staged vs unstaged diffs, and ahead/behind indicators. Real-time updates via file watching.

⌘G Toggle Git panel

Containers at
a glance

Monitor Docker containers, images, and volumes. View Kubernetes pods, services, and deployments. Stop, restart, or remove containers. Detect docker-compose projects automatically.

Powered by open source

LumaTerm is built with these incredible open-source projects.

Tauri

Desktop app framework. Native Rust backend, tiny binaries, no Electron.

xterm.js

Terminal emulator component. Powers VS Code's terminal and many others.

React

UI framework. Declarative component model for the entire interface.

portable-pty

Cross-platform pseudo-terminal. Spawns and manages shell processes in Rust.

Zustand

State management. Lightweight, fast, and minimal boilerplate.

Rust

Systems language. Memory-safe, blazing fast, powers the entire backend.

macOS Installation Note

Your download has started. Since LumaTerm is not signed with an Apple Developer certificate, macOS may show "App is damaged and can't be opened".

To fix this, run the following command in Terminal:

xattr -cr /Applications/lumaterm.app
